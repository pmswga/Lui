Steps to create the lui.exe
Install pyinstaller

Change the pip dir and typing to the Window (or Linux) cmd (or terminal) command "pip install pyinstaller"

Create the executable lui.exe file
Change the lui dir, typing the command "pyinstaller lui.py"
Get fun
Now you can use lui.exe
